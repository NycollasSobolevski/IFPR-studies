# HOW RUN
### Requisites
- [PHP](https://www.php.net/downloads.php)
- [Laravel](https://laravel.com/)
- [Composer](https://getcomposer.org/)

Install dependencies
```bash
$ composer install
```

Generate artisan key
```bash
php artisan key:generate
```

run backend 
```bash
$ php artisan serve
```

run frontend with vite
```
$ npm run dev 
```
